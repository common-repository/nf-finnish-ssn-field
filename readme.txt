This plugin extends the plugin NinjaForms with a Finnish SSN Field. You are responsible for the security of your SSN's and possible damages.
To use this field, if you haven't already, you must first install NinjaForms. Then you just add the Finnish SSN Field, like any other NinjaForms field.
